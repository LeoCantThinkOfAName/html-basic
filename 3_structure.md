# Declaration
```HTML
<!DOCTYPE html>
<html lang="en">
</html>
```

# Head
```HTML
<head>
    <title>Title</title>
</head>
```
# Body
```HTML
<body>
    Hello
</body>
```

# Meta?
```HTML
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Title</title>
</head>
```

# Basic
```HTML
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Title</title>
    </head>
    <body>
        Hello
    </body>
</html>
```

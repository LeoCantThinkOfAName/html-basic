# Value
- Color name
    - `black`
    - `gray`
    - `red`
    - `green`
    - etc...
- Hexadecimal (Hex)
```CSS
/* black */
#000000
/* white */
#ffffff
/* red */
#FF0000
```
- Decimal (RGB)
```CSS
/* black */
rgb(0,0,0)
/* white */
rgb(255,255,255)
/* red */
rgb(255,0,0)
```
- HSL
```CSS
/* black */
hsl(0, 0%, 0%)
/* white */
hsl(255, 100%, 100%)
/* red */
hsl(0, 100%, 50%)

```
- Decimal with alpha channel (RGBA)
```CSS
/* black */
rgba(0,0,0, 0.5)
/* white */
rgba(255,255,255,0.5)
/* red */
rgba(255,0,0,0.5)
```
- Transparent keyword
```CSS
background: transparent;
background: rgba(0, 0, 0, 0);
```

- Gradient
```CSS
background: linear-gradient(to top, #000, #fff);
radial-gradient(circle, #fff, #000);
```

# How to Apply
- Inline style
```HTML
<div style="background: red">Hello</div>
```
- Style tag
```HTML
<style>
    div {
        background: red;
    }
</style>
```
- Link tag
```HTML
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.2.2/dist/css/bootstrap.min.css" rel="stylesheet">
```

- @Import
```CSS
@import url("style.css");
```
